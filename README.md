# RobotMotorsControl

## Общее описание проекта
RobotMotorsControl — учебный проект для демонстрации работы с системой контроля версий Git.
Проект представляет собой простой скрипт управления двигателями робота с возможностью
изменения скорости и её плавной регулировки.

---

## Описание работы с Git

### Создание и клонирование репозитория
Репозиторий был создан на платформе GitHub. После этого он был клонирован на локальный
компьютер с помощью команды `git clone`.

### Процесс сохранения изменений
Для сохранения изменений использовались стандартные команды Git:
- `git add` — добавление файлов в индекс
- `git commit` — фиксация изменений с комментариями
- `git push` — отправка изменений в удалённый репозиторий

### Взаимодействие с ветками
Для разработки новой функциональности была создана отдельная ветка
`feature-smooth-speed`. В ней реализованы улучшения, после чего ветка
была слита с основной веткой `main`.

### Пул-реквесты
Был создан pull request для слияния ветки `feature-smooth-speed` с веткой `main`.
В рамках pull request были добавлены новые функции управления скоростью двигателей.
После проверки изменения были успешно объединены с основной веткой.

---

## Основные изменения и улучшения в коде
- Реализована функция плавного изменения скорости двигателя
- Добавлена обработка пользовательского ввода
